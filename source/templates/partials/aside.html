Smart Amp

A wireless audio amplifier for home audio distribution and automation

This site is a developer journal that features technical writeups, software theory, amateur circuit designs, opinions in programming, build progress updates and pictures, as well as the occasional embedded techno or house track.

I hope this will be a starting point for an open source hardware project. For more information on Smart Amp, please read my introduction post.
